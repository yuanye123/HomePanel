## 蓝牙操作类

## BtJniApi类 说明
#### get_ty_bt_mod
```
external fun get_ty_bt_mod():Int
```

**功能说明**

获取是否打开蓝牙能力

**返回值**

| **返回值** | **说明**       |
| ---------- | -------------- |
| 0    | 蓝牙能力打开       |
| 非0     | 蓝牙能力失败 |
